# Packageton-npm

Packegeton-npm is a skeleton structure which helps you quick start the creation of a npm module.

## Starting the application

install dependencies:

    $ make install
    
Start:

    $ make start

## Running tests

Linting:

    $ make analyse-code
    
Complete delivery:

    $ make delivery
    
## Running other tasks

See:

    make help
    
for more info.
